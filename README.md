# Digital Land brownfield land datasette

An experiment in packaging the [brownfield land collection](https://github.com/digital-land/brownfield-land-collection) as a sqlite3 database for [datasette](https://github.com/simonw/datasette).

# Building

We recommend working in [virtual environment](http://docs.python-guide.org/en/latest/dev/virtualenvs/) before installing the python dependencies:

    $ make init
    $ make
